# GITGUD

> Practice. Build. Improve.

Workspace belajar backend berbasis AI yang membantu developer meningkatkan kemampuan melalui challenge dunia nyata.

---

## Tentang Project

GITGUD adalah platform belajar backend yang menggabungkan latihan coding, AI, dan progress tracking dalam satu workspace.

Berbeda dengan platform belajar yang berfokus pada video atau artikel, GITGUD mendorong developer untuk belajar melalui praktik langsung menggunakan studi kasus nyata.

Project ini juga menjadi media pembelajaran saya dalam membangun aplikasi full-stack menggunakan React, Go, PostgreSQL, dan Google Gemini.

---

## Tujuan Project

Project ini dibuat untuk:

- Mempelajari backend development menggunakan Go.
- Memahami implementasi REST API.
- Menerapkan autentikasi menggunakan JWT.
- Mengintegrasikan AI menggunakan Google Gemini API.
- Menggunakan PostgreSQL sebagai database relasional.
- Membangun aplikasi full-stack dari tahap perencanaan hingga deployment.
- Menjadi portfolio software engineering.

## Filosofi Desain

GITGUD mengusung prinsip:

> **Low Noise, High Function**

Prinsip tersebut diterapkan melalui:

- Tampilan minimalis.
- Fokus pada konten.
- Sedikit warna aksen.
- Navigasi sederhana.
- Pengalaman layaknya IDE modern.

Inspirasi desain berasal dari:

- Linear
- GitHub
- Vercel
- Cursor IDE
- Raycast
