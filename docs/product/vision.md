# GITGUD Vision

Version: 1.0
Status: Draft
Last Updated: YYYY-MM-DD

---

# 1. Vision

Menjadi developer practice workspace yang membantu developer meningkatkan kemampuan melalui latihan yang konsisten, terarah, dan didukung AI.

---

# 2. Mission

Membantu developer belajar melalui praktik nyata, bukan hanya membaca teori.

GITGUD dirancang agar pengguna dapat menghasilkan challenge secara instan, menyelesaikannya, mendapatkan feedback dari AI, dan melacak perkembangan belajar dalam satu workspace.

---

# 3. Product Identity

GITGUD bukan LMS.

GITGUD bukan platform video pembelajaran.

GITGUD adalah AI-powered Developer Practice Workspace.

Fokus utama aplikasi adalah practice, feedback, dan progress.

---

# 4. Target Users

Primary

- Mahasiswa Informatika
- Junior Backend Developer
- Self-taught Developer
- Bootcamp Graduate

Secondary

- Frontend Developer
- Fullstack Developer
- Developer yang ingin interview preparation

---

# 5. Core Philosophy

Practice > Reading

Challenge > Course

Workspace > Classroom

Progress > Completion

Learn by Building.

---

# 6. Problem Statement

Banyak developer menghabiskan waktu menonton tutorial dan membaca dokumentasi, tetapi kurang memiliki tempat untuk berlatih secara konsisten.

Platform belajar yang ada umumnya berfokus pada materi, bukan latihan.

GITGUD hadir untuk menjembatani kesenjangan tersebut melalui AI-generated challenges dan progress tracking.

---

# 7. Value Proposition

Dengan GITGUD pengguna dapat:

- Generate challenge menggunakan AI
- Berlatih kapan saja
- Mendapat feedback
- Melihat perkembangan skill
- Membangun kebiasaan belajar
