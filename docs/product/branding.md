# GITGUD Branding

Version: 1.0

---

# Brand Overview

Name

GITGUD

Tagline

Practice. Build. Improve.

Description

GITGUD is an AI-powered developer practice workspace designed to help developers improve through real-world challenges instead of passive learning.

---

# Brand Personality

GITGUD should feel like:

- Professional
- Minimal
- Technical
- Calm
- Focused
- Modern

GITGUD should NOT feel like:

- Playful
- Childish
- Overly Gamified
- Corporate
- Social Media

---

# Brand Values

Practice First

Learning happens by solving problems.

Developer First

Every feature should improve the developer experience.

Low Noise

Remove unnecessary UI and distractions.

High Function

Every component must have a purpose.

Consistency

Keep design, interaction, and language consistent.
