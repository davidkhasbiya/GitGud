# Product Requirements Document (PRD)

Version: 1.0

---

# Product Name

GITGUD

---

# Product Summary

GITGUD is an AI-powered developer practice workspace designed to help developers improve backend skills through AI-generated challenges and continuous progress tracking.

---

# Objectives

The MVP should allow developers to:

- Sign in securely
- Generate AI challenges
- Solve challenges
- Receive AI feedback
- Track learning progress

---

# User Stories

As a developer,

I want to generate backend challenges

So that I can practice consistently.

---

As a developer,

I want my progress to be saved

So that I can monitor my improvement.

---

As a developer,

I want AI feedback

So that I know what should be improved.
